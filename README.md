# advanced ts

## ts.config

- **removeCommets**: removes comments from the compiled js file

```json
{
  "compilerOptions": {
    "removeComments": true
  }
}
```

- **strictNullChecks**: when set to true, it will not allow null or undefined to be assigned to a variable

```json
{
  "compilerOptions": {
    "strictNullChecks": true
  }
}
```
